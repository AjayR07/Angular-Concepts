• Normal Text : {{name}}
  • Lowercase Pipe : {{name|lowercase}}
  • Uppercase Pipe : {{name|uppercase}}
  • Titlecase Pipe : {{name|titlecase}}
  • Slice Pipe : {{name|slice:3}}
  • Slice Pipe : {{name|slice:3:7}}
  • Json Pipe : {{data|json}}
  • Number Pipe
  • 1.232 | number: '1.2-3 : {{ 1.232 | number: '1.2-3'}}
  • 1.232 | number: '2.4-5' : {{ 1.232 | number: '2.4-5'}}
  • 1.232 | number: '1.1-2' : {{ 1.232 | number: '1.1-2'}}
  • Percent Pipe
  • 0.25|percent : {{0.25 | percent}}
  • Currency Pipe
  • 0.25|currency : {{0.25 | currency}}
  • 0.25|currency:'GBP' : {{0.25 | currency:'GBP'}} Refer ISO Currency Codes
  • 0.25|currency:'GDB':'code' : {{0.25 | currency:'GBP':'code'}}
  • Date Pipe
  • Date : {{date}}
  • date | date : 'short' : {{date | date : 'short'}}
  • date | date : 'shortDate' : {{date | date : 'shortDate'}}
  • date | date : 'shortTime' : {{date | date : 'shortTime'}}
  • Similarly we have medium and long
  • One thing to remember is Pipe changes the data only on the view ( not the actual one )
